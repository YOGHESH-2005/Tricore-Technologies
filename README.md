# 🌐 Tricore Technologies Website

A simple, modern, and responsive website built for **Tricore Technologies**.  
It showcases company services, branding, and contact details in a clean and user-friendly way.

🔗 Live Website: https://tricore-technologies.vercel.app/

---

## 📌 About the Project

This project is a static business website created using HTML, CSS, and JavaScript.  
It is designed to present company information in a professional and responsive layout.

---

## ✨ Features

- Clean and modern UI design  
- Fully responsive layout (Mobile, Tablet, Desktop)  
- Smooth navigation between sections  
- Services showcase section  
- Image-based content sections  
- Contact information display  
- Lightweight and fast performance  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript
